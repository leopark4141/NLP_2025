# NLP_2025
파이썬활용자연어처리2025 가을학기 수업
0번 그래프
이 그래프는 텍스트 전체의 문장 길이 분포를 보여준다. 문장 길이는 텍스트의 구조적 복잡성과 정보량을 파악하는 데 중요한 요소다. 문장이 짧을수록 정보량이 제한적일 수 있고, 반대로 문장이 길면 더 많은 의미 요소가 포함된다.
우리 팀의 주제인 NLP 텍스트 분석에서 문장 길이는 전처리 과정에서 반드시 확인해야 하는 부분이며, 모델이 문장을 처리할 때 난이도에 영향을 준다. 다른 팀이 단순히 분포만 설명할 가능성이 있다면, 우리는 문장 길이가 임베딩 품질이나 모델 성능과 연결된다는 점까지 언급해 차별점을 만들 수 있다.

11번 그래프
이 그래프는 텍스트에서 자주 등장하는 단어들을 시각적으로 보여준다. 단어 빈도 분포는 텍스트의 핵심 주제나 반복적으로 강조되는 개념을 파악하는 데 중요한 역할을 한다.
우리 팀의 주제와 관련해서, 특정 단어가 지나치게 많이 등장하면 모델이 그 단어에 과도하게 반응하는 현상이 생길 수 있다. 즉, 데이터 편향이나 stopwords 처리 여부 등을 파악하는 데 유용하다. 단순히 “이 단어가 많이 나온다” 정도로 끝나는 것이 아니라, 이런 빈도 패턴이 모델 성능과 설명 가능성에 어떤 영향을 미칠지까지 해석하는 점이 차별화 요소가 될 수 있다.

17번 그래프
문서들 사이의 의미적 유사도를 cos similarity로 표현한 그래프다. 색의 농도 차이를 통해 어떤 문서들이 서로 비슷한 내용을 다루고 있는지 확인할 수 있다.
우리 팀 주제는 텍스트 의미 분석이기 때문에 이 그래프는 핵심적인 역할을 한다. 문서 간 유사도가 높다면 해당 그룹이 비슷한 주제나 서술 방식을 사용하고 있다는 뜻이며, 반대로 유사도가 낮으면 두 텍스트 집단이 명확하게 구분된다는 의미다. 이를 통해 향후 분류 모델을 만들 때 난이도나 데이터 품질을 예측할 수 있다는 점이 중요한 해석 포인트이다.

14번 그래프
문장 임베딩을 2차원으로 축소한 시각화 그래프다. PCA를 통해 문서나 문장의 의미적 구조가 어떻게 분포되어 있는지 확인할 수 있다.
우리 팀 주제와 직접적으로 연결되는 부분은, PCA로 나타난 분포를 보면 두 주제가 의미적으로 분리되는지, 아니면 서로 겹치는지를 확인할 수 있다는 점이다. 이 분석은 단순 시각화가 아니라, 이후 모델링 과정에서 분류가 쉬운 데이터인지 어려운 데이터인지까지 판단할 수 있게 해준다. 다른 팀과의 차별점은 PCA 결과를 단순 요약하는 데 그치지 않고, 이를 모델 성능 예측과 데이터 구조 해석에까지 확장한다는 점이다.
